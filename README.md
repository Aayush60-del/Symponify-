# Aura Music

A MERN music streaming starter with a warm minimal UI.

## Structure

- `client/` Vite + React frontend
- `server/` Express + MongoDB backend

## Run

```bash
cd server
npm install
cp .env.example .env
npm run dev
```

```bash
cd client
npm install
npm run dev
```

## Seed songs

```bash
cd server
npm run seed
```
